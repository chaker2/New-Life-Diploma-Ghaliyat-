دبلومة الحياة الجديدة — غاليات (نسخة الويب)
============================================

نسخة ويب خفيفة من تطبيق "دبلومة الحياة الجديدة (غاليات)" الأصلي.
HTML + CSS + JavaScript فقط — بدون أي خطوة بناء.

التشغيل:
  - افتحي index.html مباشرة في المتصفح، أو
  - ارفعي المجلد بالكامل على أي استضافة ثابتة (Netlify / Vercel / GitHub Pages / Cloudflare Pages).

الملفات:
  index.html      الصفحة الرئيسية (RTL، عربي)
  style.css       التصميم الأنثوي (وردي / لافندر / روز جولد)
  script.js       البيانات الكاملة + المنطق + مودال الفيديو
  documents/      دليل الدارسات + البرنامج التفصيلي (PDF)
  assets/         مجلد للصور (فارغ — تستخدم بدائل من الإنترنت)

  • 25 فيديو على يوتيوب + قائمتا تشغيل
  • 4 تسجيلات على SoundCloud
  • 25+ نموذج Google Forms للتقييم
  • روابط Google Drive و Box.com
  • ملفات PDF: دليل الدارسات، البرنامج التفصيلي
  • منهج مُنظَّم على 7 أسابيع
